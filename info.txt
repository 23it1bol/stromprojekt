Technische Kurzübersicht

Dieses Projekt ist ein kleines Node.js/Express API‑Backend (TypeScript). Es bietet grundlegende CRUD‑Funktionen für Kunden, Zähler und Verbrauchsdaten sowie einen Excel‑Import.

Wichtige Endpunkte (Auswahl)
- `POST /api/auth/register`  → Benutzer registrieren
- `POST /api/auth/login`     → Login (bekommt JWT)
- `GET  /api/kunden`         → Alle Kunden (auth erforderlich)
- `POST /api/kunden`         → Neuen Kunden anlegen (auth erforderlich)
- `PUT  /api/kunden/:id`     → Kunde aktualisieren (auth erforderlich)
- `POST /api/import?type=kundendaten` → Excel‑Import (auth: operator|admin)

Schnellstart
1) Abhängigkeiten installieren:
```powershell
npm install
```
2) Entwicklungsserver starten:
```powershell
npm run dev
```

Konfiguration
- Umgebungsvariablen in `.env` setzen (Beispielwerte):
```
DB_HOST=localhost
DB_USER=root
DB_PASSWORD=
DB_DATABASE=stromprojekt
PORT=3000
JWT_SECRET=dein_geheimes_token
```

Import (kurz)
- Web: `POST /api/import?type=kundendaten` mit `form-data` Feld `file` (xlsx). Benötigt JWT eines Benutzers mit Rolle `operator` oder `admin`.
- CLI: `npm run import -- --file path/to/file.xlsx --type kundendaten`

Weitere Hinweise
- Uploads werden temporär in `tmp_uploads/` abgelegt und nach Verarbeitung gelöscht.
- Passwörter werden mit `bcrypt` gehasht gespeichert.
